# Chapter 03. 데이터 분석을 위한 파이썬

## 목표
데이터 분석에 자주 활용되는 파이썬의 기본 문법을 살펴본다.

## 3.1 파이썬 기본 문법

### 1. 변수
값을 저장하는 기본 단위로,
데이터 분석 과정에서 중간 계산 결과를 저장하는 데 사용된다.

- 자료형: 문자열, 정수, 실수, 불

### 2. 조건문
특정 조건을 만족할 때만 코드를 실행하기 위해 사용한다.
데이터 분석에서는 조건에 맞는 데이터만 선택할 때 활용된다.

- if / elif / else

### 3. 반복문
동일한 작업을 여러 번 반복 실행할 때 사용한다.
여러 데이터에 같은 연산을 적용할 때 유용하다.

- for문
- while문

### 4. 함수
반복되는 코드를 하나로 묶어 재사용하기 위해 사용한다.
분석 과정을 함수로 정리하면 코드 가독성이 좋아진다.

## 3.2 파이썬 데이터 구조

### 1. 리스트
여러 값을 순서대로 저장할 수 있는 변경 가능한 데이터 구조
데이터를 추가, 삭제, 수정할 수 있다.

### 2. 튜플
순서가 있지만 변경할 수 없는 데이터 구조
변경되지 않아야 하는 데이터 묶음에 적합하다.

### 3. 딕셔너리
키와 값을 쌍으로 저장하는 데이터 구조
컬럼명과 값처럼 데이터를 빠르게 찾는 데 사용된다.

### 4. 세트
중복을 허용하지 않는 데이터 구조
데이터의 중복 여부를 확인할 때 활용된다.

## 헷갈렸던 점
- 데이터 구조의 선언 형식 기억해두기
- 리스트와 튜플은 변경 가능 여부가 가장 큰 차이
- 딕셔너리의 키가 고유한 값, 키를 통해 접근
